## 技术栈

#### 后端：

- ##### Python + Flask

- ##### 数据库采用 MySQL

#### 前端

- ##### vite + Vue 3 + yarn

- ##### UI 采用 **_Naive UI_**

- ##### 文本编辑器采用 wangEditor 5
